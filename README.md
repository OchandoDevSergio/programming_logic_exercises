# Ejercicios de lógica de programación con JS

Este repositorio consiste en un conjunto de problemas de lógica de programación que he ido resolviendo. Esporádicamente se irán añadiendo nuevos ejercicios en la medida en que se vayan realizando.

## Funcionamiento

Para poder ejecutar el código de las soluciones en los ejercicios que no incluyen ningún prompt, simplemente habría que abrir el archivo correspondiente en un editor de código y correr el código.

En los ejercicios cuya resolución incluya algún prompt, habría que entrar en el archivo index.html e introducir la dirección del archivo en que se encuentra el ejercicio que queremos ejecutar, como valor de la propiedad *src* en la línea de código 11; para a continuación abrir el archivo index.html con *Live server* o equivalentes.
